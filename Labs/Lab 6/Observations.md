This is a clock that the hour, minutes, and seconds can be adjusted.

- The buttons in the middle are used to update the hour, minutes, and seconds.
    Left-middle button (V1): hour
    Middle button (R15): minutes
    Right-middle button (R11): seconds
- The bottom button (R17) represents the reset button. The reset value is 23:59:45.
- The first switch (P5) changes the frequency (1s/0.1s).
- The 2nd-4th buttons (P4, P3, P2) set up the tens.
- The 5nd-8th buttons (R2, M4, N4, R1) set up the units.

Example:
If the switchs are X0011000 and the midle button is pressed, the minutes will change to 18.
